Digital Twin enabled Integrated Multiplatform Automation System For Industrial Assembly And Automated Storage And Retrieval System

Introduction

Created a fully automated assembly line that includes three main stages: Components Intake, Assembly, and ASRS (Automated Storage and Retrieval System). The system optimizes efficiency and product quality by employing 6 DOF Cobots, pneumatic actuators, and computer vision technology for defect identification.

OPC UA

In this project, the OPC UA server plays a crucial role as a central communication bridge, ensuring seamless interoperability among diverse components. Operating within the GUI control, PLCs, and other system elements, it facilitates standardized data exchange, real-time communication, and secure connectivity. This ensures a smooth and efficient information flow across the integrated multi-platform automation system, contributing to the project's success

Bechoff PLCs

TwinCAT, a software platform developed by Beckhoff Automation, empowers real-time automation and control, garnering widespread acclaim in machine control and industrial automation domains. Its versatile applications extend beyond industrial PCs to encompass programmable logic controllers (PLCs) and various automation systems, making TwinCAT a preferred choice for programming, configuring, and controlling diverse automation setups.

Pneumatics

In the context of this project, a comprehensive integration of pneumatic systems is observed across multiple operational stations. Specifically, the material dispense station, pinning station, and assembly station leverage the functionality of pneumatic cylinders under the control of 5/2 DCVs. The cohesive utilization of these pneumatic components plays a pivotal role in ensuring the fluid and efficient automation of diverse processes within the project, contributing to its overall success.

Digital Twin

The digital twinning in this project is greatly aided by MATLAB, which allows for the smooth import of the 3D model of the integrated system. MATLAB is used to create Simulink Simscape Multibody models that faithfully simulate the behavior of a physical system in a virtual setting. Real-time data from the physical system is compared with the digital twin using MATLAB's computational capabilities, allowing for accurate simulations..

Android App

The user-friendly app provides a smooth ordering process and lets users tailor their orders to meet particular needs. Customers can choose convenient payment options, such as cash on delivery or safe online payment, and can even specify the delivery location. After placing an order, customers can track their product's progress through various stages of production and see real-time updates on its progress. This easy-to-use application guarantees transparency and improves the ordering process while giving customers control and insights into the manufacturing process of their personalized products.

User Interface

The GUI control in this project is based on C# in Visual Studio Code, which provides a strong and user-friendly interface. The advanced C# libraries and the extensive toolkit of Visual Studio Code make it easier to create responsive controls that let users monitor system status, issue instructions, and get real time feedback. This connection improves the overall functioning and accessibility of the automation system by guaranteeing an effective, aesthetically pleasing, and user-friendly experience.
